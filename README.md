# Curriculum Vitae - Koorosh Komeili Zadeh

My personal CV builder tool. Here you can find, how to make your own CV using this project.

Install using:
```bash
npm install
```

Modify `cv-data.yaml` to update your CV content, then run:
```bash
npm start
```

The CV builder will generate html and pdf formats.

Tip: Use `**text**` in `cv-data.yaml` to make text bold in the generated CV
